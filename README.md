# EncryptedNotepad 🔒📝

**EncryptedNotepad** is a lightweight Windows desktop application designed for secure personal note-taking. It allows users to safely store critical information such as account credentials, financial PINs, mobile banking passwords, and other highly sensitive text. The application heavily leverages advanced Object-Oriented Programming (OOP) concepts in **Java (Swing)** and utilizes the industry-standard **AES (Advanced Encryption Standard)** encryption algorithm.

This project was developed as a **Final Group Project for the Object-Oriented Programming 2 Course** (4th Semester - Informatics Undergraduate Program, Jenderal Achmad Yani University).

---

## 🚀 Key Features

* **AES-128 Encryption & Decryption:** Secures text using a custom secret key/password generated by the user. The data remains locked unless the exact key is provided.
* **Lightweight Graphical User Interface (GUI):** Built natively using Java Swing, featuring a clean, minimal, and intuitive interface familiar to Windows users.
* **Masked Password Inputs (`*****`):** All PIN and password entry panels utilize a `JPasswordField` to protect sensitive keys from shoulder-surfing threats.
* **Protected Data Clearing (Clear Text):** The screen-clearing function is guarded by a dual PIN-verification process to prevent accidental data loss.
* **Windows File System Integration:** Seamlessly handles file storage using `JFileChooser`, enabling standard *Open File* and *Save / Save As* operations locally on your machine.

---

## 🛠️ OOP Architecture & Concepts

To adhere to solid Object-Oriented Design and the *Single Responsibility Principle*, the application codebase is strictly separated into modular components:

1. **Encapsulation:** Cryptographic details, underlying file handlers, and structural GUI components are kept highly secure using `private` access modifiers, restricting accidental exposures and exposing control only through specific internal helper methods.
2. **Modularity:** Divided into discrete classes with high cohesion:
   * `NotepadUI.java` – Orchestrates user interface rendering, layout constraints, and component event handling.
   * `CryptoManager.java` – Isolates the cryptographic engine, managing key generation protocols alongside core raw data encryption/decryption routines.
   * `FileManager.java` – Operates the application's underlying File I/O subsystem, managing reads and writes directly onto the host operating system.
